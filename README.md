# WorkSpaceManager
## Une application web pour gérer l'espace de travail

Vous permettant d'ajouter des employés en fonction d'une sélection de rôles. 
En fonction de ces rôles, il vérifie si cet employé est autorisé à entrer dans une certaine pièce, fixe des limites à la capacité de chaque pièce et affiche une bordure attirant l'attention lorsqu'il constate qu'une pièce qui ne devrait pas être vide l'est.

### Avec son interface utilisateur conviviale et son design réactif, WorkSpaceManager offre une expérience utilisateur fluide.
